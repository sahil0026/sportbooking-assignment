This is a test project for front-end developer position at SportBooking

This project assignment is for puspose of assessing knowledge and experience in front-end programming, analytics and problem-solving. Overall it aims to create a web-application that is divided into 4 programming tasks ordered by the level of complexity and covers different areas of web-development, such as:
  • HTML5 and CSS3 features
  • Javascript knowledge
  • Idiomatic programming
  • Unit Testing
  • AngularJS framework
  • Basic web application hosting
